parse()

parse(string  $string, string  $delimiter = ',', boolean  $header = true) : array

Return an associative array of the string.

For types with keys (YAML, JSON, INI) the keys should be used as array keys and the data nested appropriately. For row - column types (CSV, DAT) the first row should be interpretted as the column names and used as keys for each row.

Parameters

string $string

The string of data to parse.
string $delimiter

The delimiter to use when parsing the data.
boolean $header

Set false to parse data without a header.

Throws

\adamblake\Parse\Parser\adamblake\Parse\ParseException

Throws an exception if the string is invalid.

Returns

array —

The parsed data.
